1. Run `npx vitest run src/app/api/webhooks/inro/__tests__/route.test.ts --reporter=verbose`
2. All 6 tests should pass

### Expected result

- Malformed JSON returns 400
- Invalid payload (missing required fields) returns 400
- Opted-out contact returns 200 with `{ skipped: true, reason: 'opted_out' }`
- Duplicate message returns 200 with `{ skipped: true, reason: 'duplicate' }`
- Valid new message returns 200 with `{ reply: '...', conversationId: '...' }`
- Internal errors return 500 with `{ error: 'Internal server error' }` — no details leaked
